Introduction to PostGIS¶

Getting Started¶

  • This workshop uses a data bundle. Download and extract this data to a convenient location.

  • This workshop uses the following software:

    • PostgreSQL database, along with PgAdmin application, and command line tool psql.
    • PostGIS extension.

    For installers see the PostGIS Installation page.
